## ✨ Features

- **Terminal-style UI** — macOS-inspired window with profile displayed as JSON
- **Typewriter animations** — cycling shell commands & role descriptions
- **Live GitHub API** — real-time repo count, followers, stars & latest repos
- **Working contact form** — validates inputs and opens email client
- **Custom cursor** — smooth dot + ring with hover effects
- **Scroll reveal animations** — sections and skill bars animate on scroll
- **Active nav highlighting** — updates as you scroll through sections
- **Photo upload** — click to add your photo directly in the browser
- **Fully responsive** — works on mobile, tablet & desktop

## 🚀 Getting Started

### View locally
```bash
git clone https://github.com/SubhuPanda21/SubhuPanda21.github.io
cd SubhuPanda21.github.io
# Open index.html in your browser
open index.html
```

### Deploy to GitHub Pages
1. Push `index.html` to a repo named `SubhuPanda21.github.io`
2. Go to **Settings → Pages → Branch: main → Save**
3. Live at `https://SubhuPanda21.github.io` in ~2 minutes
